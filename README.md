# pmmp-Binary
PocketMine-MP のbinフォルダ  <br>
・Windows PHP-7.2.0 RC5 <br>
・Linux PHP-7.2.0 <br>

MCBE-JPN版はpocketmine.icoがMCBE-JPN仕様になってます<br>
使用の際は**bin**にリネームしてください。

# 対応start.cmdファイル
中身的にはpmmpさんのものと同じになります
